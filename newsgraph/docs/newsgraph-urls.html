Newsgraph URLs

Newsgraph is a pre-production GraphQL service that talks to CAPI via an equally new caching service called Tabula.

If you wish, you can use Newsgraph via Tabula to search for CAPI data for DevFoundry. Please be aware that these are services are in the process of being integration and load tested for production use, so problems may occur. However, we will act swiftly to fix any problems.

GraphQL documentation is available here.

Early use also allows us to field feedback from you!

We have provided five sample queries for you:

  1. Content Type queries - this is a query that can bring back individual CAPI resources for you, depending on the ID you choose to use. We've supplied sample NEWS_STORY, IMAGE, VIDEO and IMAGE_GALLERY queries. PROMO, IFRAME, CATEGORY, IMAGE_GALLERY and BLOB are also available
  2. Collection query - this is a query that can bring back a CAPI Collection via ID.
  3. Search query - this is a query that will returna v2 search in CAPI.

Beneath the sample queries you will find our V2 Schema, so you can use whichever fields you prefer.
